# Marie Hooper
Portfolio Site for Marie Hooper: Front End Developer, UI/UX Designer, UF Alumna, Florida native, Ann Arbor resident, fitness junkie, living room finance lecturer. This site is built in [Jekyll](https://jekyllrb.com/) and hosted on [Netlify](https://www.netlify.com/).

- [Breakpoint](http://breakpoint-sass.com/)
- [Reset CSS](http://meyerweb.com/eric/tools/css/reset/)

#### Fonts
- [Roboto](https://fonts.google.com/specimen/Roboto)

#### Javascript
- [jQuery](https://jquery.com/)
- [Modernizr](https://modernizr.com/download/?mq-prefixes-setclasses-teststyles)
